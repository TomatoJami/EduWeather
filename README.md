# EduWeather

Минимальный стартовый скрипт, который запрашивает текущую погоду из Open-Meteo и выводит её в консоль.

## Запуск

1. Скопируй `.env.example` в `.env` и при необходимости измени `CITY`, `LATITUDE` и `LONGITUDE`. Скрипт читает `.env` автоматически.
2. Запусти:

```bash
npm start
```

Скрипт пока только читает погоду и печатает результат. Его удобно позже расширить отправкой сообщения в Discord и добавлением логирования для Coolify/Grafana.
